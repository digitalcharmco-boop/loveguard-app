# LoveGuard - AI-Powered Relationship Safety Analysis

## Quick Start

### 1. Install Dependencies
```bash
pip install streamlit==1.28.1 openai==0.28.1 stripe==6.7.0 python-dotenv==1.0.0
```

### 2. Configure API Keys
Edit `.env` file with your keys:
```env
OPENAI_API_KEY=sk-your-actual-key-here
STRIPE_SECRET_KEY=sk_test_your-actual-key-here
STRIPE_PUBLISHABLE_KEY=pk_test_your-actual-key-here
```

### 3. Run Locally
```bash
streamlit run app.py
```

### 4. Deploy to Google Cloud
```bash
# Set environment variables
set OPENAI_API_KEY=your-key
set STRIPE_SECRET_KEY=your-key
set STRIPE_PUBLISHABLE_KEY=your-key

# Run deployment script
deploy.bat
```
